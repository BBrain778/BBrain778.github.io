平台介紹：Dify

Dify 是一個基於大型語言模型（LLM）的應用平台，讓開發者與非技術背景的使用者都能輕鬆建立 AI 助手，結合 prompt 設計、知識庫管理、API 串接與資料儲存等功能。

我完成的兩個小專案

1. 出國秘書-結合line bot

剛好最近要出國，我透過RAG的技術，將所有旅遊相關的資訊丟進知識庫(ex:航班資訊、違禁品資訊)，最後將整個專案串接line bot，如此一來我出國時候就可以直接透過line bot詢問相關問題

2. 身體健康數據顧問

我設計了一個健康顧問助手，使用者可以輸入自己的血壓、血糖與心跳資料，這些數據會儲存在 Notion 資料庫中。Dify 能根據這些資料給出飲食建議、生活習慣建議，並回答與健康相關的問題。

這門課我學到的重點

  • 如何設計 Prompt 並優化 LLM 的回應內容
  • 使用 Dify 建構 AI 助手並整合知識庫
  • 資料儲存與 API 串接實作（如與 Notion、Google Sheets 整合）
  • 基本的 HTML + CSS 排版與美化
